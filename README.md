# FocusForge 📚

> AI-powered exam tracker and study planner built in C++ — made for [Hack Club Flavortown](https://flavortown.hackclub.com/)

FocusForge helps students track upcoming exams, generate personalised AI study schedules, and receive automatic email reminders — all backed by a custom C++ server with no external hosting dependencies.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📅 **Exam Tracker** | Add exams with subject, date, difficulty, and reminders |
| 🤖 **AI Study Plans** | Day-by-day schedules generated by Claude AI or smart local algorithm |
| 📧 **Email Reminders** | Automatic SendGrid emails 1 day / 3 days / 1 week before each exam |
| 🌍 **5 Languages** | English, Spanish, French, Greek, Italian — switchable on any page |
| 🔐 **Auth** | Username/password (SHA-256) + Google OAuth via Firebase |
| 🌙 **Dark Mode** | Full dark theme, persists across sessions |
| 📊 **Dashboard Stats** | Live counts, next exam countdown, completion tracking |
| 💾 **Persistent Plans** | Study plans saved per user in SQLite, available on any device |

---

## 🛠️ Tech Stack

- **Backend:** C++ with [Crow](https://crowcpp.org/) HTTP framework
- **Database:** SQLite3 (via C API)
- **Auth:** SHA-256 password hashing (OpenSSL) + Firebase Google OAuth
- **Email:** SendGrid API via [cpr](https://github.com/libcpr/cpr) (libcurl wrapper)
- **AI:** Claude API (Anthropic) — with local fallback algorithm
- **Frontend:** Vanilla HTML/CSS/JS — zero frontend frameworks
- **Build:** CMake + vcpkg

## 🚀 Running Locally

### Prerequisites

- C++17 compiler (clang++ or g++)
- [vcpkg](https://github.com/microsoft/vcpkg)
- CMake ≥ 3.15

### 1. Clone the repo

```bash
git clone https://github.com/panayiotissavva8-dev/focusforge.git
cd focusforge
```

### 2. Install dependencies via vcpkg

```bash
vcpkg install crow sqlite3 openssl cpr nlohmann-json
```

### 3. Create `secret.env`

```env
SENDGRID_API_KEY=your_sendgrid_key_here
ANTHROPIC_API_KEY=your_anthropic_key_here   # optional
PORT=5001
```

### 4. Build and run

```bash
cmake -S . -B build \
  -DCMAKE_TOOLCHAIN_FILE=~/vcpkg/scripts/buildsystems/vcpkg.cmake \
  -G "Unix Makefiles"

cmake --build build --target focusforge

./build/focusforge
```

Open [http://localhost:5001](http://localhost:5001)

## 🌐 API Routes

| Method | Route | Description |
|---|---|---|
| GET | `/dashboard_api` | Load all exams for current user |
| POST | `/dashboard_api` | Add a new exam |
| PUT | `/dashboard_api/:id` | Update an exam |
| DELETE | `/dashboard_api/:id` | Delete an exam |
| GET | `/study_plan_api` | Load saved study plan |
| POST | `/study_plan_api` | Save / overwrite study plan |
| DELETE | `/study_plan_api` | Clear study plan |
| POST | `/login_api` | Username/password login |
| POST | `/register_api` | Register new user |
| POST | `/google_api` | Google OAuth login/register |

## 🤖 AI Study Plan

When the backend AI route is enabled, FocusForge sends a structured prompt to Claude (Anthropic) and receives a JSON study schedule. If the API is unavailable, a local scheduling algorithm takes over automatically:

- Prioritises exams by proximity (soonest first)
- Respects your study window (e.g. 10:00 AM – 9:00 PM)
- Alternates subjects to avoid burnout
- Adjusts topics based on days remaining (fundamentals → mock exams)
